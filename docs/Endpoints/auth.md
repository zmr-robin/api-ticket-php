# Auth Endpoint

## Functions

### GET

| Function      | Description |
|---------------|-------------|
| [/auth/]()            | Get all keys          
| [/auth/{id}/data]()   | Get data from key {id}

### POST

| Function      | Description |
|---------------|-------------|
| [/auth/]()    | Auth user         

### PUT

| Function      | Description |
|---------------|-------------|
| [/auth/{id}/duration]()       | Change duration of a Key {id}   

### DELETE

| Function      | Description |
|---------------|-------------|
| [/auth/{id}/delete]()       | Delete a auth key
