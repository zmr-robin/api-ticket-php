# Attachments Endpoint

## Functions

### GET

| Function      | Description |
|---------------|-------------|
| [/attachments/]()             | Get all attachments
| [/attachments/{id}]()         | Get data of attachment {id}

### POST

| Function      | Description |
|---------------|-------------|
| [/attachments/]()             | Create a new attachment

### DELETE

| Function      | Description |
|---------------|-------------|
| [/attachments/{id}]()             | Delete attachment {id}
