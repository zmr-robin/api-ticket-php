# Roles Endpoint

## Functions

### GET

| Function      | Description |
|---------------|-------------|
| [/roles/]()                   | Get all roles 
| [/roles/{id}]()               | Get role {id} data 
| [/roles/{id}/level]()         | Get role trust level

### POST 

| Function      | Description |
|---------------|-------------|
| [/roles/]()                 | Create a new role 

### UPDATE 

| Function      | Description |
|---------------|-------------|
| [/roles/{id}/level]()       | Change role trust level
| [/roles/{id}/name]()        | Change role name

### DELETE

| Function      | Description |
|---------------|-------------|
| [/roles/{id}]()               | Get role {id} data
