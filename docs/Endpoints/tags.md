# Tickets Endpoint

## Function 

### GET

| Function      | Description |
|---------------|-------------|
| [/tickets/]()                 | Get all (open) tickets 
| [/tickets/{id}]()             | Get ticket {id} data
| [/tickets/{id}/email]()       | Get email of ticket {id} creator
| [/tickets/{id}/tags]()        | Get tags of ticket {id} ticket
